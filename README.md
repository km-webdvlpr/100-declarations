# 100 Declarations

A mobile-friendly declaration tracker for daily use across phone and laptop.

## Live App

This repo is designed for GitHub Pages. The app entry point is `index.html`.

## Access

Open the GitHub Pages URL on mobile or laptop. On mobile, use "Add to Home Screen" or "Install App" for easier throughout-the-day access.

Progress is stored locally in the browser on each device. Use Backup and Restore in the app to move progress between devices.

## Spreadsheet Log

Use `100-declarations-log-template.xlsx` as the simple 100-day log database.

In the app, use `CSV` to export declaration events, then import/paste those rows into the workbook's `Declaration_Log` sheet.
